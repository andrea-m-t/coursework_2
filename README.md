## What this assignment is
This assignment covers database normalization up to 3NF, schema/table creation in MySQL, data insertion, and SQL querying using the `department` and `employee` tables.

## How to import the `.sql` file into MySQL (phpMyAdmin)
1. Open phpMyAdmin (`http://localhost/phpmyadmin`).
2. Create a database named `coursework_2` (or select an existing one).
3. Click the `Import` tab.
4. Choose the file `exported_database.sql`.
5. Click `Go` to run the script.
6. After import, run the insert statements from `Part-3_Data_Insertion.md` to populate the tables.
